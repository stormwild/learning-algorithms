# learning-algorithms

Learning Algorithms

## Pre-requisites

- Ubuntu 14
- Python 2.7.6
- PHP71

## Setup

Install PHP7.1

```
sudo add-apt-repository ppa:ondrej/php
sudo apt-get update
sudo apt-get install -y php
sudo apt-get install -y \
             libapache2-mod-php \
             php-fpm \
             php-mysql \
             php-memcached \
             php-pear \
             php-xml \
             php-mbstring \
             php-xdebug \
             php-intl \
             php-curl \
             php-gd \
             php-zip \
             php-json
```

Check version

```
php --version
```
